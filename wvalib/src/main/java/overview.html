The Digi WVA Android Library makes it easy to interact with a Wireless Vehicle Bus Adapter (WVA) from an Android application.

The starting point for using this library is to {@link com.digi.wva.WVA instantiate the WVA class}. Using methods available on the WVA object, you can, among other things:

  • {@link com.digi.wva.WVA#setTime Set the time} on the WVA;
  • {@link com.digi.wva.WVA#getConfiguration Query} and {@link com.digi.wva.WVA#configure set} configuration values;
  • Query {@link com.digi.wva.WVA#fetchVehicleData vehicle} and {@link com.digi.wva.WVA#fetchFaultCode fault code} data;
  • Subscribe to {@link com.digi.wva.WVA#subscribeToVehicleData(String, int, com.digi.wva.async.WvaCallback) vehicle} and {@link com.digi.wva.WVA#subscribeToFaultCodes fault code} data;
  • Create alarms on {@link com.digi.wva.WVA#createVehicleDataAlarm(String, com.digi.wva.async.AlarmType, float, int, com.digi.wva.async.WvaCallback) vehicle} and {@link com.digi.wva.WVA#createFaultCodeAlarm fault code} data;
  • and {@link com.digi.wva.WVA#connectEventChannel(int, com.digi.wva.async.EventChannelStateListener) connect to the TCP event channel}.
